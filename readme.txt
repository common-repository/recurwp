== Description ==

RecurWP plugin is used to integrate your Recurly account in WordPress. You can easily get recurring payments in your Recurly account using this plugin.

This plugin provides an interface with multi-step on-site checkout, so users will not be sent to a Recurly website for checkout.

Features in RecurWP
<ul>
 <li>Multistep Checkout</li>
<li>Intuitive Dashboard</li>
<li>Embed checkout form anywhere by using simple shortcode.</li>
<li>On-site Chckout</li>
</ul>
<strong>*** PRO Version ***</strong>

<ul>
 <li>Cutomer Dashboard</li>
<li>Manage Subscription from site (cancel or change)</li>
<li>Update CC info form dashboard</li>
<li>Multiple Plans</li>
<li>Shortcode Generator</li>
<li>Login Restriction</li>
<li>Customize Colors from plugin options</li>
<li>Default page generator</li>
</ul>

<strong>RecurWP Checkout </strong> [Demo](https://recurlywp.photontechs.com/checkout/ "WordPress plugin for Recurly")

== Installation ==

Navigate to WordPress > Plugins
Click Install Plugins and search for RecurWP in the search field.
Activate the newly installed plugin by clicking the Activate button.

== Frequently Asked Questions ==

= How can I setup API key? =

Please check the following documentation to generate the API key:
https://docs.recurly.com/docs/api-keys

= Can we manage subscription from dashboard? =

Yes, if you are using the pro version, this plugin does come with a user dashboard.

= Where I can see user subscriptions? =

When users checkout from the site it automatically creates a subscription on Recurly.
